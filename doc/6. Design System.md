# FreelancerOS – Design System (MVP)

## Colors

- Primary: #0F172A

- Background: #FFFFFF

- Muted: #F1F5F9

- Border: #E2E8F0

- Text Primary: #0F172A

- Text Secondary: #64748B

- Accent (Blue): #3B82F6

---

## Typography

- Font: Inter

- Page Title: 24–28px

- Section Title: 18–20px

- Body: 14–16px

- Small Text: 12–13px

---

## Spacing

Use consistent spacing scale:

4, 8, 12, 16, 24, 32

Common usage:

- Page padding: 24px
- Section gap: 24–32px
- Component gap: 16px

---

## Components (Using shadcn/ui)

- Button
- Input
- Select
- Table
- Card
- Badge
- Modal (Dialog)

---

## UI Principles

- Keep layout clean and minimal
- Use spacing instead of heavy borders
- Limit colors (mostly grayscale + one accent)
- Maintain consistent alignment
